01 — Trigger (How You Start)
🔗

📸
Screenshot
Upload a screenshot of any job post — AI reads the image directly
02 — AI Processing Pipeline
1
🕷 Job Extraction
Playwright headless browser scrapes the URL (handles JS-rendered pages). For screenshots, Claude Vision reads the image text directly.
Playwright Claude Vision API LinkedIn Scraper HTML Parser
2
🧠 AI Job Parsing (Claude)
Claude reads raw job text and extracts structured JSON: job title, company, required skills, responsibilities, experience, HR email, location, work mode.
Claude 3 Opus API JSON Output Structured Prompt
3
👤 Profile + Job Matching
Your parsed resume profile, Info settings (CTC, notice period), and the extracted job description are combined. Claude identifies skill overlaps and key points to emphasize.
Your Resume Profile Your Info Settings Skill Gap Analysis
4
📄 Tailored CV Generation
Claude rewrites your resume bullets to best match the job description — reordering skills, highlighting relevant experience — while keeping all facts true. Output is a clean ATS-friendly PDF.
Claude AI Rewriter PDFKit ATS-Friendly Template 1-Page PDF
5
✉️ Personalized Email Writing
Claude writes a concise, personalized cold email to the HR — mentioning the role, your key matching skills, notice period, expected CTC, and a call to action. Adapts tone (formal/semi-formal).
Claude Email Writer Notice Period Expected CTC Company Context
6
📬 Email Dispatch + Logging
Nodemailer sends the email via your personal SMTP with the tailored CV attached. Application is logged in the database and shown on your Home dashboard.
Nodemailer SMTP CV Attached as PDF DB Logging Dashboard Update

04 — AI Prompt Flow
Prompt 1
Job Extraction
Raw HTML / OCR Text → jobTitle, company, skills[], hrEmail, location
Prompt 2
Resume Parsing (on upload)
Raw Resume Text → experience[], skills[], education[], projects[]
Prompt 3
CV Tailoring
Resume Profile JSON Job JSON User Info (CTC etc.) → Tailored CV JSON
Prompt 4
Email Writing
Job JSON Profile Summary Notice Period + CTC → Subject + Email Body
